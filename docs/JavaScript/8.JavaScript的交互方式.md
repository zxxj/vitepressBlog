## JavaScript 的交互方式

| 交互方法       | 方法说明       | 效果查看             |
| -------------- | -------------- | -------------------- |
| alert          | 接收一个参数   | 弹窗查看             |
| console.log    | 接收多个参数   | 在浏览器控制台查看   |
| document.wirte | 接收多个字符串 | 在浏览器页面查看     |
| prompt         | 接收一个参数   | 在浏览器接收用户输入 |
